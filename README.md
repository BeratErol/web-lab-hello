# 🧪 LAB İçerikleri

## ✅ LAB-1 – Semantic HTML Yapısı

* HTML5 semantic etiketleri (`header`, `nav`, `main`, `section`, `article`, `footer`) kullanıldı.
* Anlamlı içerik hiyerarşisi oluşturuldu.
* Temel sayfa iskeleti hazırlandı.

---

## ✅ LAB-2 – Erişilebilirlik & Form Yapısı

* Accessible iletişim formu eklendi.
* `label`, `aria` özellikleri ve form doğrulama yapısı düzenlendi.
* Profil fotoğrafı entegrasyonu yapıldı.
* Lighthouse analizi gerçekleştirildi.

---

## ✅ LAB-3 – Responsive Tasarım (Mobile-First)

* Mobil-first yaklaşımı benimsendi.
* CSS değişkenleri (design tokens) kullanıldı.
* `clamp()` ile fluid tipografi uygulandı.
* Header ve navigasyon Flexbox ile düzenlendi.
* Proje kartları CSS Grid ile 3 breakpoint’e uyumlu hale getirildi:

  * Mobil (varsayılan)
  * ≥ 640px (tablet)
  * ≥ 1024px (masaüstü)
* Responsive ekran görüntüleri eklendi.
* `CSS-KARARLARI.md` dosyası oluşturuldu.
